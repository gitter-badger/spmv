spmv
====
[![Gitter](https://badges.gitter.im/Join Chat.svg)](https://gitter.im/elrrrrrrr/spmv?utm_source=badge&utm_medium=badge&utm_campaign=pr-badge&utm_content=badge)
> 解决spm多个版本之间的平滑切换

##安装

```shell
npm config set prefix /usr/local
npm i -g spmv
```

##使用

```shell
spmv install
spmv use
```

##查看帮助

```shell
spmv -h
```
